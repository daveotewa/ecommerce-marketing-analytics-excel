# Ecommerce Marketing Analytics System (Excel)

## Project Overview
This project is an end-to-end ecommerce marketing analytics system built in Excel.  
It analyzes marketing performance across Meta Ads, Google Ads, and Shopify sales data.

The system transforms raw marketing data into actionable insights for campaign scaling and performance monitoring.

---

## Key Features

• Automated data cleaning and transformation  
• Dynamic date filtering via control panel  
• Campaign performance analysis  
• Creative performance evaluation  
• Product performance insights  
• Decision framework (Scale / Watch / Kill)  
• Executive dashboard with KPIs and revenue trends  

---

## Dashboard Metrics

The dashboard provides quick insight into:

- Total Spend
- Total Revenue
- Total Purchases
- Overall ROAS
- Top Performing Campaign
- Top Selling Product
- Revenue Trend
